**Step 1**: int loan = 49800;

**Step 2**: double interestRate = 0.05;
- 실수형 변수 선언

**Step 3**: int interest = (int)(loan * interestRate);
- **캐스팅 개념** (핵심!)
- 컴파일 에러 처리
- 최종 결과: 2490벨

## 💻 IDE 동작 원리

### 코드 검증 방식
1. **사용자 입력**: textarea에서 Java 코드 작성
2. **버튼 클릭**: "▶ 코드 실행" 버튼
3. **검증 로직**: 
   - 필수 키워드 포함 확인 (int, double, (int) 등)
   - 올바른 형식 검사
4. **출력**: 성공/실패 메시지
5. **해금**: 성공 시 "다음 단계" 버튼 활성화

### 예시 검증

**Step 1 성공:**
```
✅ 성공! int 타입을 올바르게 사용했습니다.

너굴: "그렇지! int를 썼구나. 깔끔해!"
```

**Step 1 실패:**
```
❌ 오류: int loan = 49800; 형태로 선언해야 합니다.
```

**Step 3 (캐스팅) 실패:**
```
❌ 컴파일 에러: Type mismatch: cannot convert from double to int

너굴: "캐스팅이 필요해! (int)를 붙여줘!"
```

**Step 3 성공:**
```
첫 달 이자: 2490벨

✅ 완벽해! 캐스팅을 올바르게 사용했습니다.
```
