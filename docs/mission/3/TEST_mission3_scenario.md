# 🧪 Mission 3 (Java Variables) 검증 시나리오

**목표**: Mission 3의 Java IDE가 정상 작동하고, 입력한 코드가 서버에서 실행되어 결과가 올바르게 출력되는지 검증한다.

## 📋 테스트 단계 (Test Steps)

1.  **접속 및 로그인**:
    *   `http://localhost:3000` 접속
    *   시작 버튼 클릭 (또는 로그인)

2.  **미션 진입**:
    *   미션 목록 또는 지도에서 **"Mission 3: 변수의 세계 (Java)"** 선택
    *   미션 페이지(`unit2` 또는 해당 경로) 진입 확인

3.  **IDE 로딩 확인**:
    *   좌측: 미션 가이드와 너굴 사장님 대화창 확인
    *   우측상단: 코드 에디터(Monaco Editor/CodeMirror) 로딩 확인
    *   우측하단: 실행 결과 콘솔창 확인

4.  **코드 작성 및 실행 (Happy Path)**:
    *   에디터에 아래 Java 코드 입력:
        ```java
        int apple = 5;
        System.out.println("사과 개수: " + apple);
        ```
    *   **[실행]** 버튼 클릭
    *   예상 결과: 콘솔창에 `사과 개수: 5` 출력 확인

5.  **오답 코드 실행 (Error Path)**:
    *   문법 오류가 있는 코드 입력 (예: 세미콜론 누락)
    *   **[실행]** 버튼 클릭
    *   예상 결과: 콘솔창에 `Compilation Error` 또는 관련 에러 메시지 출력

6.  **최종 제출**:
    *   정답 코드 입력 후 **[제출하기]** 버튼 클릭
    *   예상 결과: "미션 성공!" 메시지 또는 다음 단계로의 진행 확인
