## 📐 디자인 시스템

### 스마트폰 크기
- 전체 너비: 360px
- 전체 높이: 600px
- 패딩: 15px
- 모서리: 8px (border-radius)

### 타이포그래피
- 제목: 24px, 굵음
- 소제목: 18px, 굵음
- 본문: 14px, 보통
- 캡션: 12px, 얇음
- 코드: 12px, 모노스페이스

## 🎬 애니메이션 효과

### 1. 탭 전환
- 이전 탭 (오른쪽으로 슬라이드 아웃)
- 새로운 탭 (왼쪽에서 슬라이드 인)
- 속도: 300ms, 이징: ease-in-out

### 2. 너굴 반응
1. 너굴 나타남 (위에서 내려옴, 200ms)
2. 메시지 나타남 (페이드인, 200ms)
3. 3초 동안 표시
4. 사라짐 (아래로 내려가며, 200ms)

### 3. 포인트 획득
- 숫자 애니메이션: 0 → 500점 (1초)
- 색상 변화: 흰색 → 주황색
- 크기: 원래 크기 → 1.2배 → 원래 크기
